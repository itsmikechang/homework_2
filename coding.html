Javascript Quiz App

An app designed to test one's general knowledge in JavaScript. The Quiz is 10 questions long and the user has 150 seconds to complete it. When a user completes a question, it will let the user at the bottom of the next question if the user has gotten the previous question correct or not. When the user completes the quiz, the app will take them to the scoreboard submision page where they can enter their name into the score board. In the scoreboard page, the user has the option to try the quiz again or clear their previous score

Work Day Schdeuler

An app designed to schedule out a user's entire work day. The app consists of 9 rows, each designating a hour of the work day. The user in puts their task for that specific hour and hits the save button next to the input box. The entry is stored into local storage and when the user refreshes, the previously save entry will display until the user erases the entry and hits save.

nomstop', a food app for the indecsive milennial.

A simple food app that takes in 3 user input parameters (price, cuisine and radius (based on user location)). When the user runs the app with all the inputted parameters, the app will aggregate the results, randomize them and give the user only one resturant that is local to them (based on user input radius). The app will display the randomized results business info, a picture from yelp and a map of where the business is at.
